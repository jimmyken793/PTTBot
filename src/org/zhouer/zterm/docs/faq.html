問：我該如何設定用 Firefox 當外部瀏覽器？
答：在 選項 -> 偏好設定 -> 一般 -> 外部瀏覽器啟動命令 設定：

Windows:
C:\Program Files\Mozilla Firefox\firefox.exe "%u"
Linux:
firefox %u

問：我該怎麼複製或彩色複製？
答：請見使用說明的「複製」與「貼上」部份。

問：meta 鍵是鍵盤上的那個鍵？
答：在 Windows/Linux 下是 Alt 鍵，Mac 下是 Apple 鍵。

問：好像不支援全形字偵測？
問：為什麼有些設定是被 diable 的？
答：因為還沒有寫好，未來的版本會改進。

問：為什麼介面是英文的？
答：請將系統語言設定為中文：

Windows:
「控制台」->「地區及語言選項」->「標準及格式」->中文（台灣）
Linux:
設定 LANG 環境變數為 zh_TW.UTF8 或 zh_TW.Big5
