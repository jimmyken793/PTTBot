連線：
  • 在位置列輸入 telnet://hostname:port 或是 ssh://hostname:port
  • 若省略 port 部份，則 telnet 預設是 23 port, ssh 預設是 22 port
  • 若省略 telnet:// 或是 ssh:// 部份，則預設是 telnet
  • 可直接輸入名稱或是別名連線，名稱與別名可在站台管理中設定
快速鍵：
  • meta-d: 切換到位置列
  • meta-t, meta-s: 快速輸入 telnet:// 與 ssh://
  • meta-c, control-insert: 複製
  • meta-v, shift-insert: 貼上
  • meta-w: 斷線、刪除分頁
  • meta-r: 重新連線
  • meta-q: 開新連線
  • meta-left, meta-z: 上一分頁
  • meta-right, meta-x: 下一分頁
  • meta-0 ~ meta-9: 快速切換分頁
  • meta-,: 開啟偏好設定
  • meta-.: 開啟站台管理
  • 註：meta 在 Windows/Linux 下是 Alt 鍵，Mac 下是 Apple 鍵
滑鼠：
  • click: 若是 http url 則開啟連結
  • double click: 選取連續字元
  • triple click: 選取整行
複製：
  • 預設狀態：
    • 以滑鼠選取待複製文字
    • 使用快速鍵 meta-c 或 control-insert 即可複製
    • 點擊工具列或是視窗選單的複製或彩色複製
    • 滑鼠選取時按住 meta 鍵或 ctrl + meta 可立即複製或彩色複製
  • 選取後立即複製：
    • 滑鼠選取文字後放開立刻複製
    • 滑鼠選取時按住 ctrl 鍵則彩色複製
    • 若單純選取但不想複製則在選取時按住 meta 鍵
貼上：
  • 使用快速鍵 meta-v 或 shift-insert 即可貼上
  • 點擊工具列或是視窗選單的貼上或彩色貼上
  • 單擊滑鼠中鍵也可以貼上，同時按著 ctrl 則會彩色貼上
